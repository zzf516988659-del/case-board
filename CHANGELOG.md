## [0.3.22] - 2026-06-18

> **非诉「合同起草」上线** + 非诉改卡片入口 + **法律工具新增两个计算器(交通事故赔偿 / 劳动解除赔偿,带法律依据弹窗)**,并带出上一批改动。
> 新功能均标 **Beta**,生成内容与计算结果请自行核对。本版三平台齐发(Mac Apple Silicon / Intel + Windows)。
> 合同起草方法论借鉴开源项目 [pa1nrui1/legal-skills](https://github.com/pa1nrui1/legal-skills)(作者「小潘律师」,MIT)—— 感谢分享;prompt / 引擎 / 导出均自建。
> 感谢 [@gcheng001](https://github.com/gcheng001) 贡献多后端配置修复(PR #15)。

### ✨ 新增
- **非诉「合同起草」(Beta)**:描述交易需求 + 选我方立场 → AI 按「三观四步法」判合同类型、按「三点一线法」结构起草 → 引导补全要素 → 生成完整合同草案,可导出 Word。支持多轮修订 + 版本管理(版本历史 / 修订生成下一版 / 标记最终版 / 草案库)与起草偏好库(按合同类型自动套用条款偏好)。
- **非诉模块卡片化**:非诉 tab 改成卡片式功能入口(合同审查 / 合同起草各一张卡),为后续功能留位。
- **法律工具新增两个计算器(Beta,均带「法律依据」弹窗)**:
  - **交通事故赔偿计算器**:残疾/死亡赔偿金(年龄折减 + 伤残系数)、被扶养人生活费、丧葬费、各项费用 + 责任比例 + 交强险/商业险扣减;地区标准填本地现行值。
  - **劳动解除赔偿计算器**:经济补偿 N / 代通知金 N+1 / 违法解除 2N + 3 倍社平封顶 + 12 年封顶 + 未休年假工资。
- **辅助在线立案:运行环境一键检测 + 安装**。

### 🔧 优化 / 修复
- 多 LLM 后端(GLM / MiMo / 自定义 OpenAI 兼容)各自独立存配置、`/v4` 端点归一,切换后端不再串配置(整合 [@gcheng001](https://github.com/gcheng001) PR #15)。

## [0.3.14] - 2026-06-12

### 🆕 应用内自动更新
- 发现新版本后可「立即更新」:应用内下载(带进度条)→ 自动安装 → 重启,不用再手动下载覆盖。
- **安全**:每个更新包都由作者私钥签名,App 内置公钥强制验签,签名不对直接拒装;私钥不在开源代码内,仅作者本机 + CI 加密 secret 持有。即便源码公开、下载站被攻破或遭中间人,攻击者拿不到私钥就签不出能装的包。
- 应用内更新不可用 / 失败时自动回退到手动下载;升级重启后会提示本次更新内容。
- 注:本次需手动更新一次(旧版没有更新组件),从 0.3.14 起后续版本才能一键更新。

## [0.3.13] - 2026-06-12

### 🆕 新增(0.3.10 ~ 0.3.13 累积同步)

- **OCR 双线路**:新增 PaddleOCR VL-1.6(百度 AI Studio)云端识别,设置页填访问令牌(免费 20,000 页/天)即自动成为 MinerU 的备用线路 — 失败 / 排队超时 / 额度用完自动切换;也可一键设为主力。不填则一切照旧。
- **审级模型**:一个案件可记多个审级(仲裁 → 一审 → 二审 → 再审),各审级独立案号 / 承办机关 / 当事人称谓;首页显示最新审级,详情页新增「审级历程」;案件状态新增「仲裁中」「再审中」。
- **团队版(无服务器)**:创建团队 → 配对码加入 → 同一办公网内自动互相同步案件进度(接力传播);权限逐人配置,有编辑权可改队友案件状态,改动留痕可撤销。只同步案件登记信息,不传文档原文。
- **外部数据源接入(MCP)**:元典 / 企查查等平台的云端 MCP 服务可整段粘贴配置接入 AI 助手。
- **还款自动入账**:银行转账截图识别出还款后自动记进执行页还款记录(带 AI 标记,可删)。
- 法院短信可按当事人姓名匹配案件;案件详情页新增「重新分析」按钮;全案分析阶段显式进度提示,失败透传真实错误。
- **Windows 支持**:安装包内置 WebView2 引导器(构建工作流见 `.github/workflows`)。

### 🔧 变更

- 更新提醒每个新版本只弹一次;详情页手改信息首页同步更新。

### 🆕 新增

- **Word 导出排版升级**:统一到原生 OOXML 排版引擎,表格不再散架,支持仿宋 / 黑体 / 方正小标宋字体。
- **一个文件夹自动识别为多个案件**:拖入文件夹时自动检测,提供拆分预览确认,共用材料挂到每个案件。

### 🔧 变更

- 文档导出抬头补回案号 / 法院 / 案由等元信息。

### ⚠️ 已知限制

- 共用材料目前对每个案件各抽取一次(尚未做抽取结果复用)。
